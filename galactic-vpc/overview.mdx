## Datum’s virtual backbone service

Datum is working to provide a global, virtual backbone service we call “Galactic VPC”. Soon, you’ll be able to create your own Galactic VPC and use it for traffic routing and connections.

Galactic uses Segment Routing over IPv6 (SRv6) to apply advanced policy and traffic control to packets. Network devices process these instructions to forward packets along a specific path. Each instruction is an IPv6 address called a Segment Identifier (SID).

Using this capability, Datum can program traffic paths using the native IPv6 data plane built into all modern routers and kernels, without the need for MPLS. You can read more about SRv6 in [RFC 8986](https://www.rfc-editor.org/rfc/rfc8986.html).
