## How to Use

1. Copy the [Project Template](/projects/template)
2. Create a new file: `projects/<project-name>.mdx`
3. Add it to `docs.json` under the Projects group
4. Fill in goals, milestones, and linked GitHub resources

<Tip>
  Use the **GitHub Issues** section to embed live issue references. See [Issue Workflow](/github/issue-workflow) for how to keep them in sync.
</Tip>
