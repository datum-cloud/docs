## Context

Our platform aims to provide traditional internet-scale primitives (e.g. intelligent edge proxies, private backbones, cloud onramps, peering, meet-me-rooms, etc) but with an experience that is 100% programmatic, agent-first, and backed by AGPLv3 open source.

We believe that a network cloud in the AI era is a bit different than a traditional CDN, mainly because the use case is intelligence not caching. Additionally, we think that open (open source, open ecosystems) is a critical requirement in a world that is increasingly fragmented and regulated.

Our target customer is a modern service provider, which we lovingly call [Alt Clouds](https://www.alt-cloud.org/). We're looking to support 1000's of them, so neutrality is an important feature.
