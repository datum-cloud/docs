### Synopsis

Create a resource from a file or from stdin.

 JSON and YAML formats are accepted.

```

```

### Examples

```

  kubectl create -f ./pod.json

  # Create a pod based on the JSON passed into stdin
  cat pod.json | kubectl create -f -
```
