Our first "Connector" is a flexible Tunnel powered by the Iroh protocol.  

Iroh is a modular, Rust-based networking stack designed to establish fast, direct, and secure peer-to-peer (P2P) connections between devices. It functions as a modern alternative to traditional client-server networking by allowing devices to "dial" each other directly using public keys, bypassing NATs and firewalls, rather than relying on IP addresses.

Our first use case for Iroh Tunnels is with our [desktop apps](https://www.datum.net/download/), which help you securely expose localhost to the internet. Future iterations will include "anywhere to anywhere" Tunnels.
