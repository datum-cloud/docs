We are currently building a public cloud with a generous "forever free" tier, and working closely with design partners on managed cloud and BYOC approaches.

Our public cloud is opinionated, but powered by key partners. We call these our "house brand" services:

- **AI Edge** - An envoy edge proxy and AI gateway, powered by Tetrate
- **Connectors** - Various connections, starting with QUIC tunnels, powered by Iroh
- **Essentials**: DNS, domains, metrics export, activity logs, collaboration, etc.

In flight:

- **Uniworkers** - Millisecond cold-start, isolated compute, powered by Unikraft
- **Galactic VPC** - Virtual private backbones that span anywhere, powered by SRv6
- **vMMR** - A virtual "meet me room"
