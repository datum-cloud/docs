## How to Use

1. Copy the [Customer Template](/customers/template)
2. Create a new file: `customers/<company-name>.mdx`
3. Add it to `docs.json` under the Customers group
4. Fill in account details, use cases, and notes

<Tip>
  Link GitHub issues to customer pages using the issue URL in the **Open Issues** section of each customer page.
</Tip>
